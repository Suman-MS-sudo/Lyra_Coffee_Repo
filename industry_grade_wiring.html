Industry Grade Wiring Reference — ESP32 + 3-Channel Temperature Control + Water Level + Relay Control

Safety Requirements — Read Before Wiring

  • MCB 10A on every 240V Live line
  • RCCB 25A 30mA for earth leakage
  • Earth all metal enclosures
  • IP65 rated enclosure box
  • 1.5mm min wire for 240V lines
  • Ferrule crimps on all wire ends
  • DIN rail terminal blocks
  • Never touch 240V while powered
  • SSR-25DA on heatsink always
  • Fuse on each heater circuit
  • Label all wires with markers
  • Test with multimeter before power
  • Neutral never through relay
  • E-stop button on panel
  • Pilot lamps for each zone
  • Separate ESP32 5V SMPS
