Safety — Read First

  • Always put 10A fuse on 230V Live before relay
  • Never touch 230V wiring while powered
  • Earth/ground all metal enclosures
  • Use 10A rated wire for 230V lines
  • SSR-25DA gets hot — mount on heatsink

ESP32 → JQC-3F Relay

  • GPIO27 → IN (active-LOW: LOW = relay ON)
  • 5V → VCC
  • GND → GND
  • COM ← 230V Live (after fuse)
  • NO → DTT-72 Live input

DTT-72 Connections

  • L ← JQC-3F NO (switched Live)
  • N ← 230V Neutral (direct)
  • IN+ ← Thermocouple + (red)
  • IN− ← Thermocouple − (blue)
  • OUT+ → SSR-25DA PIN 3
  • OUT− → SSR-25DA PIN 4

SSR-25DA Connections

  • PIN 3 (+) ← DTT-72 OUT+
  • PIN 4 (−) ← DTT-72 OUT−
  • PIN 1 (AC) ← 230V Live (direct)
  • PIN 2 (AC) → Heater wire 1
  • Heater wire 2 → 230V Neutral

MAX6675 (Optional)

  • Only needed if ESP32 must read temperature
  • DTT-72 controls heater independently
  • CLK → GPIO18, DO → GPIO19, CS → GPIO5
  • VCC → 3.3V, GND → GND

DTT-72 Front Panel Setup

  • Set sensor type to K (thermocouple)
  • Set SV (set value) to your target temp
  • Control action: ON-OFF
  • ESP32 only switches power — not setpoint
  • Allow 3s boot time after power-on
